# Reflection

## What I built
- A file-driven job-hunting agent that reads job posters, resumes, and KB notes.
- Automated reports: job analysis, skill-gap report, tailored resume suggestions, interview questions.
- Application tracker integration with reminders and a status dashboard.
- Extra features: reminder urgency labels and a cover letter draft.

## What I tested
- Ran the agent with sample inputs in input_jobs/, input_resumes/, and input_kb/.
- Verified outputs are created in outputs/ and tracker/reminders.txt.
- Confirmed reminders include urgency labels based on dates in applications.csv.

## Improvements and uniqueness
- Menu-based file selection when multiple job posters or resumes exist.
- Reminder urgency tags (TODAY, TOMORROW, THIS WEEK, OVERDUE).
- Cover letter draft generator for quick outreach.

## Tools used
- GitHub Copilot for drafting and refining code and documentation.
